UserProfileDelegate

public protocol UserProfileDelegate

A protocol for UserProfile manipulation. The current default implementation only copies UserProfile displayName, id, and provider. In case this default behaviour is insufficient, this protocol should be implemented, and passed as an option to the corresponding plugin’s constructor.
